# Flip RN Dev Assigment

## Setup
Install dependencies.
```bash
$ npm install 
```

Install and link Cocoapods dependency
```bash
$ npx pod-install ios
```

## Build 
Build the project

IOS
```bash
$ npm run ios -- --no-packager
```

Android
```bash
$ npm run android -- --no-packager
```

## Running
```bash
$ npm start
```

Then open the app
